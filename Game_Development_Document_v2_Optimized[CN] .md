## 1. 棋盘地图布局设计

### 设计理念
棋盘分为四条边，对应 Dota 2 地图的逻辑区域：
1. **底边：** 天辉优势路 & 野区 (Radiant Safe Lane & Jungle)
2. **左边：** 河道 & 中路 (River & Mid Lane)
3. **顶边：** 夜魇劣势路 & 野区 (Dire Offlane & Jungle)
4. **右边：** 高地 & 遗迹 (High Ground & Ancients) —— **地价最贵区域**

### 2.3 建筑建造规则

#### 同色系地块要求
```
2块地产：可以开始建造绿房子
3块地产：建造速度更快（可直接建2栋）
4块地产：建造成本-10%
```

#### 建造顺序
```
1. 必须拥有同色系全部地块
2. 必须在所有地块上平均分配房子（不能跳级）
3. 例如：不能在A地建3栋房子，而B地只有1栋
```

#### 红旅馆建造条件
```
1. 同色系所有地块都有4栋绿房子
2. 支付500G
3. 拆除4栋绿房子，升级为1个红旅馆
```

---

### 2.4 破产与复活机制（新增）

#### 破产判定
```
当玩家无法支付租金/税费/罚款时，进入破产流程：
1. 自动出售所有建筑（按建造成本的50%回收）
2. 拍卖所有地产（起拍价=购买价的80%）
3. 仍无法支付 → 破产出局
```

#### 买活机制（Buyback）
```
【条件】
- 每位玩家游戏中只能买活1次
- 需要支付500G"复活费"

【效果】
- 清空所有地产和建筑
- 回到起点，获得1000G重新开始
- 保留英雄和已购买的道具

【战术意义】
- 给予翻盘机会
- 延长游戏时间
- 增加戏剧性
```

### 3.2 英雄技能详细说明

#### 🟢 平衡良好的英雄

**炸弹人 (Techies)**
```
【埋雷】
- 玩家回合开始时，可选择在任意地块埋雷
- 下一个踩中的玩家：
  ∟ 晕眩1回合（跳过下一次行动）
  ∟ 支付100G给炸弹人
- 地雷会在棋盘上显示小红点（但不显示具体位置）
- CD：4回合
```

**斧王 (Axe)**
```
【反击螺旋】
- 被动触发：当其他玩家移动到斧王所在格子时
- 效果：强制对方支付50G"摩擦费"
- 特殊互动：配合帕吉的肉钩使用
```

**帕吉 (Pudge)**
```
【肉钩】
- 可以钩取前方3-4格内的任意玩家
- 钩到自己的地产上 → 对方需支付租金
- 钩到斧王所在格 → 触发反击螺旋
- 战术玩法：配合队友或自己的高价地产
- CD：4回合
```

#### 🟡 需要策略运用的英雄

**炼金术士 (Alchemist)** ⚖️ 已平衡
```
【贪婪】（被动）
- 工资加成：每次经过泉水额外 +30G（总计230G）
- 租金加成：所有收取的租金 +5%
- 30回合预计额外收入：~450G工资 + 300G租金 = 750G
- 定位：稳健发育型，后期经济优势明显
```

**赏金猎人 (Bounty Hunter)** ⚖️ 已平衡
```
【忍术】（被动）
- 踩中他人地块时，少付20%租金
- 对方仍收取100%租金（差额由银行补贴）
- 每圈最多触发3次（忍术能量限制）
- 定位：生存能力强，但不影响对手收入
```

**先知 (Nature's Prophet)** ⚖️ 已平衡
```
【传送】（主动）
- 可传送到：①任意未被购买的地块 ②自己拥有的地块
- 传送费：50G
- CD：7回合（从5回合延长）
- 定位：灵活控图，但有经济成本
```

**虚空假面 (Faceless Void)** ⚖️ 已平衡
```
【时间漫游】（主动）
- 掷骰子后，如果不满意结果，可以撤销
- 选择后退1-3格（玩家自选距离）
- CD：5回合（从3回合延长）
- 定位：风险控制，精准避险
```

## 8. 游戏流程与规则

### 8.1 游戏开始流程

```
1️⃣ 选择玩家数量（2-5人）
2️⃣ 每位玩家选择英雄（10选1，不可重复）
3️⃣ 决定回合顺序（所有玩家掷骰子，点数大的先手）
4️⃣ 每位玩家获得起始资金：2000G
5️⃣ 所有英雄棋子放置在"泉水"起点
6️⃣ 游戏开始！
```

---

### 8.2 单回合流程

```
🎲 回合开始阶段
├─ 检查是否在监狱（是→尝试出狱；否→继续）
├─ 道具购买阶段（可选）
└─ 技能冷却-1

🎯 行动阶段
├─ 掷2颗骰子
├─ 移动棋子
├─ 触发格子效果
│  ├─ 空地→购买选项
│  ├─ 他人地产→支付租金
│  ├─ 机会/中立物品→抽卡
│  └─ 特殊格→特殊效果
└─ 英雄技能使用（可选）

🏁 回合结束阶段
└─ 回合数+1，切换到下一位玩家
```

---

### 8.3 特殊规则

#### 双数规则
```
【掷出双数时（两个骰子点数相同）】
- 执行完当前回合后，立即再掷一次骰子
- 连续3次双数→强制入狱（防止无限循环）

【在监狱时】
- 掷出双数→免费出狱
- 未掷出双数→继续关押或支付50G保释
```

#### 工资领取
```
【经过或停留在"泉水"】
- 基础工资：200G
- 炼金术士额外：+30G
- 怨灵系带额外：+50G
- 最高可达：280G/圈
```

#### 破产拍卖
```
【玩家破产时】
1. 自动出售所有建筑（建造成本的50%）
2. 拍卖所有地产
   - 起拍价 = 购买价的80%
   - 其他玩家竞拍
   - 最高价者获得
3. 仍无法支付→出局或买活
```

---

### 8.4 胜利条件

```
【标准胜利】
- 其他所有玩家破产出局
- 最后剩余的玩家获胜

【时间胜利】（可选规则）
- 达到50回合时，资产最多者获胜
- 资产 = 现金 + 地产价值 + 建筑价值 + 道具价值

【遗迹胜利】（可选规则）
- 拥有天辉遗迹+夜魇遗迹+两者都有红旅馆
- 立即获胜
```
